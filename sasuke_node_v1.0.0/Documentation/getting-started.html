Prerequisites

To get started with Tapeli, please follow the steps below to install and set up all the necessary prerequisites

  • Node.js - v16x or v18x

    To use our build tools and take full advantage of Tapeli's customization options, you will need to download and install Node.js. If you do not have Node.js v16.x or v18.x installed already, you can download the package installer from the official website. We recommend that you download the stable version of Node.js (LTS) to ensure the best possible experience.

    Download Node.js
  • Yarn

    Ensure that you have Yarn installed and running on your computer. If you already have Yarn installed, you can skip this step.

Installation:

To install all the required dependencies for the template, open your terminal, navigate to the folder where you have the Tapeli files, and run the command yarn install. This will install all necessary dependencies in the node_modules folder. Once the installation is complete, you are ready to run Tapeli for the first time.

You can run the following commands to run projects locally: Or build for production use:

Command Description
gulp It will compiles source files and monitor changes to your demo code, including JavaScript and Sass.
yarn dev It will start development server. The development server is accessible at http://localhost:8000/.

You can start development server on different port by editing port in config.env file.

If you encounter any issues or have any questions during the setup process, please don't hesitate to reach out to us via email. We are always here to help and would be glad to assist you in any way we can.
