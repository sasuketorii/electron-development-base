Introduction

Hando Node JS is a fully featured premium admin template built on top of awesome Bootstrap 5.3.3 and Node Js. It's crafted using modern web technologies HTML5, SCSS, Node JS and JavaScript. The theme includes beautiful dashboard, many ready to use hand crafted components, widgets, pages, etc. Ideal for Web Apps, Dashboards, Admin Panels, and SaaS interfaces, features vite based build tools, SCSS variables base, and RTL support. Quickly customize layouts and modes using data attributes without extensive code changes. Enjoy a beautiful UI with minimal coding effort.

It's a powerful and highly flexible solution, ideal for web applications such as CRM, CMS, admin panels, and dashboards. Fully responsive, it looks perfect on both mobile devices and tablets. Designed with modularity in mind, it helps reduce development time and costs.

This documentation is guideline for all users who can even be a beginner or experienced web developer.
